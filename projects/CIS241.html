Welcome

This project is a dynamic PHP/MySQL web application built using a Model-View-Controller (MVC) architecture. It allows full CRUD operations for products, customers, suppliers, and categories using a MySQL database. The system is hosted locally using XAMPP and demonstrates server-side processing, database integration, and modular PHP design.

View Live Product Catalog (XAMPP Required)

Sample SQL Query

            SELECT p.name, p.description, p.price,
                s.name AS supplier,
                c.name AS category
            FROM products p
            JOIN suppliers s ON p.supplier_id = s.supplier_id
            JOIN product_categories c ON p.category_id = c.category_id;
